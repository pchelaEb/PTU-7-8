Результирующий вектор для Т-опор равен:
0 0 1 0 
Переход t1 неживой
Переход t2 неживой
Переход t3 живой
Переход t4 неживой

Таким образом в моделируемом объекте не все переходы живые

Результирующий вектор для Р-опор равен:
0 0 0 
Позиция p1 недостижима
Позиция p2 недостижима
Позиция p3 недостижима

Таким образом в моделируемом объекте не все позиции достижимы
