NEURAL NETWORK WIZARD - Возможности Neural Network Wizard

Подробности
Категория: Свободное программное обеспечение
Опубликовано 17.03.2014 10:18
Просмотров: 759
Страница 2 из 2

 

Рассмотрим возможности пакета Neural Network Wizard на следующем примере:

Создать нейронную сеть для распознавания функции «Исключающее ИЛИ». Число  еременных для входа и выхода равно соответственно 2 и 1. Данные для моделирования представлены в табл. 1.

 

1. Для начала подготовим обучающие примеры, то есть представим данные в текстовом файле с разделителем Tab, или Пробел (рис. 1).

 

Рис. 1 – Текстовый файл с обучающим множеством

Далее указываем путь к файлу, содержащему обучающую выборку (рис. 2).

 

Рис. 2 – Выбор файла с обучающим множеством

На следующем шаге определяем входные/выходные поля и их свойства, выбираем способ нормализации данных (рис. 3).

 

Рис. 3. – Определение входных/выходных полей и способа нормирования

2. На следующем этапе определяем конфигурацию нейронной сети, здесь же можно задать параметр сигмоиды (рис. 4).

 

Рис. 4. – Определение топологии сети

3. Перед началом процесса обучения созданной нейронной сети, пользователю необходимо задать скорость и момент обучения, а также выбрать условия, по которым обучение будет завершено (рис. 5).

 

Рис. 5. – Выбор критериев обучения сети

Ход процесса обучения и результаты можно наблюдать в окне, представленном на рис 6. Здесь на графиках можно отслеживать насколько результаты, предсказанные нейронной сетью, совпадают со значениями в обучающей (слева) и тестовой (справа) выборке. Каждый пример обозначен на графике точкой. Если точка попадает на выделенную линию (диагональ), то значит, сеть предсказала результат с достаточно высокой точностью. Если точка находится выше диагонали, значит, сеть недооценила, ниже – переоценила. Необходимо добиваться, чтобы точки располагались как можно ближе к диагонали.

 

Рис. 6. – Ход процесса обучения

4. После завершения обучения можно проверить качество обучения, а также использовать обученную нейронную сеть для расчетов (рис. 7).

 

Рис. 7. – Проверка результатов обучения

Neural Network Wizard позволяет сохранить все параметры и настройки в виде файла с расширением nnw. Данный файл представляет собой результат обучения нейросистемы, и позволяет восстанавливать параметры полученной нейронной сети для дальнейшего ее использования в конкретных приложениях. По своей структуре данный файл представляет собой стандартный ini–файл. Он состоит из отдельных секций, каждая из которых описывает один из этапов работы системы (подробное описание всех секций можно найти в справочной системе пакета).

Таким образом, пакет Neural Network Wizard можно использовать в задачах, для решения которых применимы обучение с учителем и сети персептронного типа, а это широкий спектр задач прогнозирования, аппроксимации функции, классификации.
