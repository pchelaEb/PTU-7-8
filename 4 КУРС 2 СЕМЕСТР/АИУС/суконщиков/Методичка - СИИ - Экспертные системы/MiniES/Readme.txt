"Малая экспертная система" версия 1.0, распространяется свободно
( MiniES v1.0 (freeware) )
Системные требования: Windows 95/98.

Программа представляет собой простую экспертную систему, 
использующую байесовскую систему логического вывода. 
Она предназначена для проведения консультации с пользователем в 
какой-либо прикладной области (на которую настроена загруженная 
база знаний) с целью определения вероятностей возможных исходов 
и использует для этого оценку правдоподобности некоторых предпосылок, 
получаемую от пользователя.
Важным достоинством данной программы является возможность создания и 
использования собственной базы знаний.

Пожалуйста, прежде чем решите обращаться с вопросами к автору программы,
прочитайте внимательно справочный файл. В нём содержится описание системы, 
проведения консультации с ней, приведён формат базы знаний, даны примеры 
и рекомендации по созданию собственной базы. Также приводится список литературы,
прочитанной автором и рекомендуемой для изучения области искусственного интеллекта, 
связанной с построением экспертных систем.
